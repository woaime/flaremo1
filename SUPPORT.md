# Support

FlareMo is an open-source project. The fastest path depends on what you need.

## Bugs

Use the bug report template:

https://github.com/realchendahuang/FlareMo/issues/new?template=bug_report.yml

Include the release tag or commit, deployment method, reproduction steps, expected result, actual result, and relevant logs or screenshots.

## Memos Compatibility

Use the Memos compatibility template:

https://github.com/realchendahuang/FlareMo/issues/new?template=memos_compatibility.yml

Include the client or tool, request path, request body, important headers, FlareMo response, and the expected Memos behavior. Do not paste Cloudflare Access secrets.

## Feature Requests

Use the feature request template:

https://github.com/realchendahuang/FlareMo/issues/new?template=feature_request.yml

Explain the real workflow first, then the proposed behavior.

## Security

Do not report private vulnerabilities in public issues. Follow SECURITY.md:

https://github.com/realchendahuang/FlareMo/security/policy

## Deployment Help

Read these first:

- README.md
- docs/deploy.md
- docs/agent-deploy.md
- docs/maintenance.md

For Cloudflare Access, D1, R2, or Wrangler problems, include the exact command and error output. Redact account IDs, Access client secrets, API tokens, and private URLs if needed.
